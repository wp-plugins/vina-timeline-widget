== Description ==

The Vina Timeline Widget for WordPress is a good choice for showing serial content to bring more efficiency transferring information to readers. Vina Timeline Widget a killer jquery timeline wordpress plugin which will allow you to give more life to the boring timelines. Vina Timeline Widget supports both horizontal and vertical layouts, and interesting thing is that you can easily specify parameters for most attributes: speed, transparency etc.

Features:

- Support multi Wordpress Version.
- Get title, image, description from post item.
- Load post items from a specific category.
- Allows to adjust the number, ordering of post item displayed. 
- Customizable speed, delay, size etc.
- Allows to show/hide title, image, description and readmore button.
- Allows resize image before display.
- Languages included: English
- Allow loading wordpress widget inside post, page.
- To manipulate almost everything !
- Totally customizable and skinnable using HTML and CSS
- Lightweight, modern and fast-loading design.
- W3C XHTML 1.0 Transitional. W3C CSS Valid.
- Tableless design and 100% CSS-based
- Fully compatible IE8+, Firefox 10+, Flock 0.7+, Netscape, Safari, Opera 9.5
- Easy installation

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire 'vina-timeline-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the 'vina-timeline-widget' to your sidebar
